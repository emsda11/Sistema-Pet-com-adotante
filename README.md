# Sistema de Adoção de Pets com Responsável

Projeto acadêmico em Java Spring Boot com API REST, HTML, CSS e JavaScript.

## Estrutura

```text
src/main/java/com/example/petadocao
├── PetAdocaoApplication.java
├── controller
│   ├── PetController.java
│   └── AdotanteController.java
├── service
│   ├── PetService.java
│   └── AdotanteService.java
└── model
    ├── Pet.java
    └── Adotante.java
```

## Como executar

1. Abra o projeto em uma IDE.
2. Aguarde o Maven baixar as dependências.
3. Execute a classe `PetAdocaoApplication.java`.
4. Acesse:

```text
http://localhost:8080/index.html
```
